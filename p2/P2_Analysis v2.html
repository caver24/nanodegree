# method used for analysis below
def extract_survival_and_non_counts(df_gb):
    #this is a somewhat specialized method that would only really work on data with survior data coded this way
    #takes in a group by object and returns a DF with 2 coloums that contain the count of survived and the count of non survived
    output = pd.DataFrame({})
    output['not_survived'] = df_gb.count()['Survived'] - df_gb.sum()['Survived']
    output = output.join(df_gb.sum()['Survived'])
    return output

def calc_ratio(df):
    df['total'] = df['Survived'] + df['not_survived']
    df['survival_ratio'] = df['Survived'] / df['total']
    return df

Conclusion¶

As noticed in the Gender Analysis, it does appear based on this evidence that more females than males survived. From these samples this looks like a 74% change of survival amung females and a 19% chance of survival among males. Now there is quite a large difference in distributions of the genders with more than 2x as many males, which could cause some skew to the results, one way to address this would be to preform a 2-proportion z-test to determine if the differnece of 2 proportions is statistically significant. As noticed in the Class Analysis, it does appear that a passenger's class came with differing levels of survival, with the 1st class having the highest rate of survival. From these samples it appears that class 1 has a survival rate of 63%, class 2 a rate of 47%, and class 3 of 24%. As noticed in the Age Analyis, it does appear that most children and the very elderly had a higher rate of survival than the rest. Due to missing data points for age, some records of this sample had to be ignored, this could reslut in a less accurate approximation based on a sample population. One final note to make is this data was only a sample of the total titanic passengers. Because we dont know how this data was obtained or have any control of its selection we cannot gaurentee it acuratly refects the total population analysis.
